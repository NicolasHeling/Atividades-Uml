1- A JVM seria o motor que roda o código Java.
 O JRM seria o ambiente onde tem as bíbliotes pra rodar.
O JDK é o kit completo  para desenvolver 	
	
2- Ela só define quais características e ações uma coisa vai ter no sistema

3-  É a coisa real e "viva" rodando na memória do computador, construída a partir da classe.
 
4- É a porta de entrada do programa. É o ponto de partida onde o Java começa a ler e executar o seu código.

5- Atributo é uma característica ou dado.
 Método é uma ação ou comportamento. 

6- Ele serve pra preparar o objeto assim que ele "nasce". Basicamente, é ele quem define os valores iniciais dos atributos bem na hora que você usa o comando new.

7- É importante para proteger os dados. Ele tranca as informações sensíveis (usando o private) para que outras partes do código não baguncem suas variáveis sem querer.
